# Williams Maikol Bouzid — Research & Engineering Portfolio

Static, responsive portfolio website built with semantic HTML, CSS and vanilla JavaScript.

## Preview locally

```bash
python3 -m http.server 8000 --directory portfolio
```

Then open `http://localhost:8000`.

## Structure

- `index.html` — content, project cards, SEO metadata and structured data
- `styles.css` — responsive visual system and motion
- `app.js` — EN/FR language switch, mobile navigation, project filters and reveal animations
- `assets/Williams_Maikol_Bouzid_CV.pdf` — downloadable CV

## Editorial method

New material is curated through a research-oriented thread: **question → method → implementation →
validation → limitation**. A document is added as a standalone project only when it provides enough
evidence for that chain; exercises and course briefs are used to refine skills and context without being
presented as research outputs.

The July 2026 content review added:

- a dedicated radar and sonar research-focus section grounded in verified academic evidence;
- radar coursework (17.3/20, ranked 3/25) and the associated DGA lecturer recommendation;
- sonar systems (18/20) and optimal filtering (17.5/20), framed as academic foundations rather than professional expertise;
- blind source separation with AMUSE, lagged covariance matrices and spectral/noise checks;
- Raspberry Pi–MultiWii integration, MSP/MAVLink communication, PWM acquisition and SITL missions;
- bridge-sensor conditioning and ADC modelling as supporting instrumentation experience.

Course PDFs, handwritten work, internal correspondence and raw project material are not published with
the website.

## Publishing

The folder can be deployed directly to GitHub Pages, Netlify, Cloudflare Pages or any static host.
